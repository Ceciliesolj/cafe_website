Changes done according to our original plan:

  1. JSfiles:
    • The JSfile structure is not the same as in P2. Now we have cart.js, drinks.js, food.js, index.js and footerAndHeader.js. This is different than the origional file plan: shoppingcart.js, slideshow.js, header.js, and maps.js.
    • We added a javascript function for handling the footer and header code, which is the same on every page. This was not something we had thought of in the earlier submissions. It is located in footerAndHeader.js
  2. HTML:
    • We added a new page for the shopping cart: cart.html and located in the html folder.
    • We decided to not show the user which page they are at in the header by having a darker color on the different pages. We decided this because it is very intuitive which page you are on.
  3. CSS:
    • We made the slideshow on the front page mobile friendly.
  4. Footer:
    • More interesting information that we originally planned. We decided to not include the name of the cafe and when it was last modified. This because it looks better without it.
    • Added links to related pages and mail application.
  5. Homepage:
    • Changed the quote and description to fit one "page" for simplicity. It was too much scrolling from what it looked like in the mockups. Since the page should be as simple as possible and users should be able to navigate around the pages in a short time, we did this change.
  6. Shopping cart icon:
    • Moved this to the navbar for easily access from everywhere on the page.
    • The pop-up function is changed, instead you are redirected to a new page displaying your goods. It makes the page look cleaner, and more information can be shown.
  7. Drinks, pop up menu:
    • Instead of a pop up menu to choose extras in the drink menu, there is now an opportunity in the shopping cart.
  8. Drinks and food sites
    • We added some content that is always apparent on the food and drink pages in order to avoid the page being completely empty upon no matches.
    • We added a note saying there are no products matching the results if the customers ticked boxes that didn't match any products.
    • We added an option to bring a lunch box as well as the cup, and this enhances the environmentally friendly aspects of CoffeeCorner.
